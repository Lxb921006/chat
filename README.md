# chatAI
这是一个聚合了主流大模型AI网站，可无缝访问如：chatgpt，claude，文心一言，讯飞星火等，持续添加中....

## 安装
```
npm install
```

### 运行
```
npm run serve
```

### 编译部署
```
npm run build
```

### node版本
v16.4.0  

### npm版本
7.18.1

### 主要功能
1.持续上下文对话, 会话保存, 删除, 恢复;  
2.代码高亮, 复制, 丝滑响应;  
3.图片生成, 文件分析;  
4.多平台ai切换:  
&emsp;&emsp;claude(已完成);  
&emsp;&emsp;chatgpt(已完成);  
&emsp;&emsp;讯飞星火(已完成);  
&emsp;&emsp;文心一言(已完成);
