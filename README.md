# chatAI

## 安装
```
npm install
```

### 运行
```
npm run serve
```

### 编译部署
```
npm run build
```


### 主要功能
1.持续上下文对话, 会话保存, 删除, 恢复;  
2.代码高亮, 复制, 丝滑响应;  
3.多平台ai切换(claude(已完成), chatgpt(已完成), 讯飞星火(正在接入), gpt-4(正在接入));
