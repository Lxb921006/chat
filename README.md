# chatAI
这是一个聚合了市面上主流大模型AI网站，方便你无缝访问如：chatgpt，claude, 文心一语，讯飞星火，第三方ai助手等

## 安装
```
npm install
```

### 运行
```
npm run serve
```

### 编译部署
```
npm run build
```

### node版本
v16.4.0  

### npm版本
7.18.1

### 主要功能
1.持续上下文对话, 会话保存, 删除, 恢复;  
2.代码高亮, 复制, 丝滑响应;  
3.多平台ai切换:
